# 10-prompt-library.md — ARCANA Prompt Library

## 1. How to Use

ใช้ Prompt เหล่านี้กับ ChatGPT, Claude, Gemini, Codex หรือ AI ตัวอื่น เพื่อสร้างคอนเทนต์ให้ ARCANA โดยให้ AI อ่านไฟล์ Knowledge Base ก่อนเสมอ

คำสั่งเริ่มต้นที่แนะนำ:

อ่านไฟล์ Knowledge Base ทั้งหมดของ ARCANA ก่อน แล้วสร้างคอนเทนต์ตาม Brand Guideline, Writing Style, Product Knowledge, Content Rules และ SEO Guideline ห้ามกล่าวอ้างเกินจริง ห้าม Hard Sell และต้องให้ความรู้ก่อนขาย
