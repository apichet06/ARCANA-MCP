# 05-seo-guideline.md — ARCANA SEO Guideline

## 1. SEO Objective

SEO ของ ARCANA มีเป้าหมายหลัก 3 อย่าง

1. เพิ่ม Organic Traffic
2. ให้ลูกค้าเข้าใจสินค้าก่อนซื้อ
3. สร้างความน่าเชื่อถือให้แบรนด์ผ่านบทความและ Landing Page

ARCANA ควรทำ SEO แบบให้ความรู้ ไม่ใช่แค่ยัด Keyword

---

## 2. SEO Content Types

### 2.1 Product Highlight

เหมาะกับบทความที่พาไปสู่การซื้อโดยตรง เช่น
- Natural Toothpaste
- ASARA Relief
- FitMe Golf Glove
- Bird’s Nest Collagen

### 2.2 Tips / Knowledge

เหมาะกับบทความให้ความรู้ เช่น
- วิธีเลือกถุงมือกอล์ฟ
- วิธีดูแลกลิ่นปาก
- วิธีดูแลตัวเองช่วงทำงานหนัก
- วิธีเพิ่มผักในมื้ออาหาร

### 2.3 Comparison

เหมาะกับบทความเปรียบเทียบ เช่น
- FitMe vs FMe BiO
- ยาสีฟันธรรมชาติ vs ยาสีฟันทั่วไป
- สเปรย์อโรม่า vs ยาหม่อง

### 2.4 Gift Guide

เหมาะกับเทศกาล เช่น
- ของขวัญสุขภาพสำหรับผู้ใหญ่
- ของขวัญสำหรับคนทำงาน
- ของขวัญสำหรับนักกอล์ฟ

## 3. Keyword Strategy

### Primary Keyword

Keyword หลักควรอยู่ใน:
- Title
- H1
- URL
- Meta Title
- Meta Description
- Intro
- อย่างน้อย 1 H2

### Secondary Keyword

ใช้กระจายตามเนื้อหาแบบธรรมชาติ

### Long-tail Keyword

ARCANA ควรเน้น Long-tail Keyword เพราะตรงกับ Search Intent มากกว่า

ตัวอย่าง:
- ถุงมือกอล์ฟสำหรับมือเล็ก
- ยาสีฟันลดกลิ่นปาก
- สเปรย์อโรม่าแก้คอบ่าไหล่
- เส้นผักสำหรับคนไม่ชอบกินผัก
- รังนกคอลลาเจนของขวัญผู้ใหญ่

## 5. Meta Title Guideline

Meta Title ควรมีความยาวประมาณ 50-60 ตัวอักษร

### Template

[Primary Keyword] | [Benefit] | ARCANA

### Example

ถุงมือกอล์ฟ FitMe กระชับมือ ใส่สบาย | ARCANA

ยาสีฟันลดกลิ่นปาก ดูแลลมหายใจมั่นใจ | ARCANA

ASARA สเปรย์อโรม่า ผ่อนคลายระหว่างวัน | ARCANA

---

## 6. Meta Description Guideline

Meta Description ควรมีความยาวประมาณ 120-155 ตัวอักษร

### Template

เรียนรู้วิธีดูแล [ปัญหา] พร้อมแนะนำ [สินค้า] ที่ ARCANA คัดสรร เพื่อช่วยให้คุณ [Benefit] ในชีวิตประจำวัน

### Example

เรียนรู้วิธีเลือกถุงมือกอล์ฟที่พอดีมือ พร้อมแนะนำ FitMe Golf Glove สำหรับคนที่ต้องการความกระชับและมั่นใจในทุกวงสวิง

---

## 7. URL Guideline

URL ควรเป็นภาษาอังกฤษ สั้น และมี Keyword

### Good

/products/fitme-golf-glove  
/blogs/product-highlights/natural-toothpaste-for-bad-breath  
/blogs/tips/how-to-choose-golf-glove

### Avoid

/blogs/post123  
/blogs/สินค้าใหม่ลดราคาแรงมาก  
/blogs/very-long-url-with-too-many-words

---

## 8. Internal Linking

ทุก Blog ควรมี Internal Link อย่างน้อย 2-4 จุด

### Link Types

1. Link ไป Product Page
2. Link ไป Blog ที่เกี่ยวข้อง
3. Link ไป Collection Page
4. Link ไปหน้า About ARCANA

### Example

บทความยาสีฟันควร Link ไป:
- Product Page ยาสีฟัน
- Blog สุขภาพช่องปาก
- Collection Self-care

บทความ FitMe ควร Link ไป:
- Product Page FitMe
- Blog FitMe vs FMe BiO
- Collection Golf

เขียนบทความ SEO สำหรับ Shopify Blog ของ ARCANA

[Keyword รอง]

[สินค้า]

[Informational / Commercial / Transactional]

กลุ่มเป้าหมาย:
[กลุ่มเป้าหมาย]

เงื่อนไข:

- ให้ความรู้ก่อนขาย
- ไม่กล่าวอ้างเกินจริง
- ใส่ Internal Link Suggestion
- ใส่ Meta Title
- ใส่ Meta Description
- ใส่ URL Slug
- ใส่ CTA ตอนท้าย

## 11. SEO Checklist

ก่อนเผยแพร่บทความ ให้ตรวจสอบ:

- มี Primary Keyword ชัดเจน
- Title น่าสนใจ
- H1 มี Keyword
- Meta Title ไม่ยาวเกินไป
- Meta Description ชัดเจน
- URL อ่านง่าย
- มี Internal Link
- มี CTA
- ไม่มีการกล่าวอ้างเกินจริง
- อ่านง่ายบนมือถือ
- มี Alt Text สำหรับภาพ
- มีคำตอบต่อ Search Intent
