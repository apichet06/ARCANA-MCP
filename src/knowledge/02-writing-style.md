# 02-writing-style.md — ARCANA Writing Style Guide

## 1. Main Writing Concept

การเขียนคอนเทนต์ของ ARCANA ต้องยึดหลัก

> ให้ความรู้ก่อนขาย  
> เข้าใจลูกค้าก่อนแนะนำสินค้า  
> ขายประโยชน์ ไม่ใช่ขายแค่คุณสมบัติ

ARCANA ไม่ควรเขียนเหมือนร้านขายของทั่วไปที่เปิดมาด้วยราคา โปรโมชัน หรือคำขายแรง ๆ แต่ควรเขียนแบบให้ลูกค้ารู้สึกว่า “แบรนด์เข้าใจปัญหาของเรา”

---

## 2. Core Framework

ใช้ Framework นี้เป็นหลักกับทุกคอนเทนต์

Problem → Empathy → Education → Product → Benefit → Trust → CTA

### Problem

เริ่มจากปัญหาที่ลูกค้าเจอจริง

ตัวอย่าง:
- รู้สึกไม่มั่นใจเวลามีกลิ่นปาก
- ถุงมือกอล์ฟหลวมจนจับไม้ไม่ถนัด
- นั่งทำงานนานแล้วคอ บ่า ไหล่ ตึง
- อยากกินผัก แต่ไม่มีเวลาเตรียมอาหาร

### Empathy

แสดงความเข้าใจ

ตัวอย่าง:
- ปัญหาเล็ก ๆ เหล่านี้ อาจส่งผลต่อความมั่นใจในแต่ละวัน
- สำหรับคนทำงานออฟฟิศ อาการตึงบ่าอาจกลายเป็นเรื่องที่เจอแทบทุกวัน

### Education

ให้ความรู้หรืออธิบายเหตุผล

ตัวอย่าง:
- กลิ่นปากอาจเกิดจากคราบแบคทีเรีย การดูแลช่องปากไม่ทั่วถึง หรืออาหารบางประเภท
- ถุงมือที่ไม่พอดีมืออาจทำให้การจับไม้ไม่มั่นคง

### Product

แนะนำสินค้าแบบนุ่มนวล

ตัวอย่าง:
- ARCANA จึงคัดสรร FitMe Golf Glove สำหรับผู้ที่ต้องการถุงมือที่กระชับและใส่สบาย
- ASARA Relief Aroma Spray เป็นอีกตัวเลือกสำหรับวันที่ร่างกายต้องการความผ่อนคลาย

### Benefit

อธิบายผลลัพธ์ที่ลูกค้าจะได้รับ

ตัวอย่าง:
- ช่วยให้รู้สึกมั่นใจมากขึ้นในการพูดคุยใกล้ชิด
- ช่วยให้จับไม้ได้มั่นคงขึ้นในทุกวงสวิง
- ช่วยเติมความสดชื่นและความผ่อนคลายในระหว่างวัน

### Trust

เพิ่มความมั่นใจ

ตัวอย่าง:
- เหมาะสำหรับใช้ในชีวิตประจำวัน
- ใช้งานง่าย พกพาสะดวก
- คัดสรรโดย ARCANA

### CTA

ปิดด้วยคำชวนที่ไม่แข็งเกินไป

ตัวอย่าง:
- เลือกดูสินค้าที่เหมาะกับคุณ
- เริ่มดูแลตัวเองง่าย ๆ ได้ตั้งแต่วันนี้
- อ่านรายละเอียดเพิ่มเติม
- เลือกสิ่งที่ดีขึ้นให้ทุกวันของคุณ

## 3. Blog Writing Style

### โครงสร้าง Blog แนะนำ

1. Title
2. Intro / Hook
3. Problem
4. Why It Matters
5. How to Choose / How to Care
6. Product Recommendation
7. Summary
8. CTA

### Blog Intro Template

เริ่มด้วยสถานการณ์จริง

ตัวอย่าง:

หลายคนอาจคิดว่ากลิ่นปากเป็นเรื่องเล็ก แต่ในความเป็นจริง ปัญหานี้ส่งผลต่อความมั่นใจมากกว่าที่คิด โดยเฉพาะเวลาต้องพูดคุยใกล้ชิด ประชุม หรือพบลูกค้า การดูแลช่องปากอย่างเหมาะสมจึงเป็นหนึ่งในพื้นฐานสำคัญของการดูแลตัวเองในทุกวัน

### Blog CTA Template

หากคุณกำลังมองหาตัวช่วยที่ใช้งานง่ายในชีวิตประจำวัน ARCANA คัดสรรสินค้าที่เหมาะสำหรับการดูแลตัวเองอย่างต่อเนื่อง เพื่อให้คุณเลือกสิ่งที่เหมาะกับตัวเองได้อย่างมั่นใจ

## 4. Landing Page Writing Style

### LP Structure

Hero → Pain Point → Why It Matters → Product Benefits → How to Use → Reviews → FAQ → Final CTA

### Hero Section

ควรมี:
- Headline ชัดเจน
- Subheadline ที่เชื่อมกับปัญหา
- Benefit หลัก
- CTA

ตัวอย่าง:

ถุงมือกอล์ฟที่พอดีมือ เพื่อความมั่นใจในทุกวงสวิง

FitMe Golf Glove ออกแบบมาเพื่อความกระชับ ใส่สบาย และช่วยให้คุณจับไม้ได้มั่นคงขึ้นตลอดเกม

CTA:
เลือกไซซ์ของคุณ

### Pain Point Section

ใช้ภาษาที่ลูกค้าเจอจริง

ตัวอย่าง:
- ถุงมือหลวมเกินไป
- จับไม้แล้วรู้สึกไม่มั่นใจ
- เหงื่อออกแล้วลื่น
- ใส่นานแล้วไม่สบายมือ

### Benefit Section

เปลี่ยน Feature เป็น Benefit เสมอ

Feature:
มีไซซ์ SS

Benefit:
เหมาะสำหรับผู้ที่มีมือเล็ก และหาถุงมือที่พอดีมือได้ยาก

Feature:
กลิ่นหอมจากอโรม่า

Benefit:
ช่วยสร้างความรู้สึกผ่อนคลายระหว่างวัน

## 5. Facebook Post Style

### Framework

Hook → Pain Point → Benefit → Product → CTA

### ตัวอย่าง

เคยไหม? แปรงฟันแล้ว แต่ยังไม่มั่นใจเรื่องกลิ่นปาก

กลิ่นปากอาจเกิดจากหลายสาเหตุ ทั้งคราบแบคทีเรีย อาหาร หรือการดูแลช่องปากที่ไม่ทั่วถึง การเลือกยาสีฟันที่เหมาะกับการดูแลช่องปากในทุกวัน จึงเป็นเรื่องสำคัญมากกว่าที่คิด

ARCANA คัดสรร Natural Toothpaste สำหรับคนที่อยากดูแลลมหายใจ เหงือก และความมั่นใจในชีวิตประจำวัน

อ่านรายละเอียดเพิ่มเติมได้ที่ ARCANA

## 7. TikTok / Reels Script Style

### Structure

0-3 sec: Hook  
3-10 sec: Problem  
10-20 sec: Explain  
20-30 sec: Product / Solution  
30-35 sec: CTA

### Example Script

Hook:
กลิ่นปากไม่ใช่เรื่องเล็ก ถ้าต้องคุยกับคนใกล้ ๆ ทุกวัน

Problem:
หลายคนแปรงฟันแล้ว แต่ยังรู้สึกไม่มั่นใจ เพราะกลิ่นปากอาจเกิดจากคราบสะสมและการดูแลช่องปากที่ไม่ทั่วถึง

Solution:
การเลือกยาสีฟันที่เหมาะกับการดูแลลมหายใจและเหงือก จึงเป็นสิ่งที่ช่วยเสริมความมั่นใจได้ในทุกวัน

Product:
ARCANA คัดสรร Natural Toothpaste สำหรับคนที่อยากเริ่มดูแลช่องปากอย่างจริงจัง

CTA:
อ่านรายละเอียดเพิ่มเติมได้ที่ ARCANA

## 8. Google Ads Style

Google Ads ต้องสั้น ชัด และเน้น Search Intent

### Headline Examples

- ถุงมือกอล์ฟ FitMe
- ถุงมือกอล์ฟพอดีมือ
- ยาสีฟันลดกลิ่นปาก
- สเปรย์อโรม่า ASARA
- รังนกคอลลาเจน
- เส้นผัก Queen Kale

### Description Examples

- คัดสรรสินค้าคุณภาพเพื่อดูแลตัวเองในทุกวัน เลือกซื้อผ่าน ARCANA
- ถุงมือกอล์ฟกระชับมือ ใส่สบาย เหมาะสำหรับผู้เล่นที่ต้องการความมั่นใจ
- ดูแลลมหายใจและสุขภาพช่องปากง่าย ๆ ด้วยยาสีฟันธรรมชาติ

## 9. Meta Ads Style

Meta Ads ควรเน้น Hook และ Pain Point

### Primary Text Template

เคยเจอปัญหา [Pain Point] ไหม?

[อธิบายปัญหาแบบเข้าใจง่าย]

ARCANA คัดสรร [Product] สำหรับคนที่ต้องการ [Benefit]

เริ่มดูแลตัวเองง่าย ๆ ได้ตั้งแต่วันนี้

### Headline Template

- ตัวช่วยดูแลตัวเองทุกวัน
- เลือกสิ่งที่เหมาะกับคุณ
- สุขภาพดี เริ่มจากสิ่งเล็ก ๆ
- มั่นใจขึ้นในทุกวัน

## 10. Email Style

Email ควรสุภาพ อบอุ่น และให้ความรู้

### Subject Examples

- เริ่มดูแลตัวเองจากสิ่งเล็ก ๆ ในทุกวัน
- เคล็ดลับดูแลสุขภาพช่องปากให้มั่นใจขึ้น
- ถุงมือกอล์ฟที่พอดีมือ สำคัญกว่าที่คิด
- วันทำงานหนัก ๆ ร่างกายคุณต้องการการผ่อนคลาย

### Email Structure

Greeting → Problem → Education → Product → CTA

## 11. FAQ Style

FAQ ต้องตอบตรง ชัด และไม่กล่าวอ้างเกินจริง

### Example

Q: ใช้ทุกวันได้ไหม?  
A: สามารถใช้ได้ตามคำแนะนำของสินค้า เหมาะสำหรับการดูแลตัวเองในชีวิตประจำวัน

Q: เหมาะกับใคร?  
A: เหมาะสำหรับผู้ที่ต้องการตัวช่วยในการดูแลตัวเอง และมองหาสินค้าที่ใช้งานง่ายในชีวิตประจำวัน

## 12. Writing Rules for AI

เวลาสร้างคอนเทนต์ให้ ARCANA ให้ทำตามกฎนี้เสมอ

1. ห้ามเริ่มด้วยการขายแรง
2. ห้ามเคลมรักษาโรค
3. ห้ามใช้คำว่า “ดีที่สุด” หากไม่มีหลักฐาน
4. ต้องเริ่มจาก Pain Point หรือ Lifestyle Insight
5. ต้องอธิบายประโยชน์ ไม่ใช่แค่คุณสมบัติ
6. ใช้ภาษาง่าย อ่านบนมือถือได้
7. CTA ต้องนุ่มนวลและน่าเชื่อถือ
8. รักษาภาพลักษณ์พรีเมี่ยม เข้าถึงง่าย
